LPMS-B2: 9-Axis IMU / AHRS / Motion Sensor with Bluetooth 2.1 and 4.1 Connectivity

Description

The LP-Research Motion Sensor Bluetooth version 2 (LPMS-B2) is a miniature wireless inertial measurement unit (IMU) / attitude and heading reference system (AHRS). The unit is very versatile, performing accurate, high speed orientation and relative displacement measurements. Through the use of three different MEMS sensors (3-axis gyroscope, 3-axis accelerometer and 3-axis magnetometer) drift-free, high-speed orientation data around all three axes is achieved. Additionally temperature and barometric pressure sensors allow accurate determination of the unit’s altitude. The LPMS-B2 communicates with a host system via a Bluetooth Classic 2.1 or Low Energy 4.1 connection. Multiple LPMS-B2 units can be used simultaneously with one host system (7 units maximum on Windows/Android). The sensor is capable of data transmission rates of up to 400Hz.
