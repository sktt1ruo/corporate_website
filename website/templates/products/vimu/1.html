Vision - IMU (VIMU)

Description

The advantageous VIMU sensor features a modular design including an imaging sensor, an image processor, an IMU module, and industrial standard interfaces of high-speed USB 3.0 and CAN. The standalone VIMU sensor provides onboard intelligent functions of image processing, depth estimation, motion estimation, and feature tracking. With the onboard image processor, the smart sensor provides core processing functions required in industrial and robot applications, and it is fully extendible with application-specific processing requirements.
