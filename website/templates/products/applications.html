Applications

LP-RESEARCH sensing and anlysis technology is used in various application fields, ranging from sport performance evaluation to medical surgery analysis. In many application cases, data fusion from multiple raw sensor data sources allows a more accurate and fast solution to a specific hard engineering problem than conventional methods.
